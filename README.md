# Ecotravel-



**Aluno:** José Lucas Fernandes Lourenço e Alessandro Sousa Felix 


## Descrição
Protótipo de aplicativo informativo sobre atrações de turismo sustentável no Brasil. Permite navegar entre atrações, ver imagem ilustrativa, alternar seções (dicas, localização, curiosidades) e mostrar mais/menos informações.


## Execução (local)


1. Instale o Expo CLI (se necessário):

2. No diretório do projeto:

3. Abra no emulador ou no app Expo Go pelo QR code.


## Observações
- Este é um protótipo informativo — não contém gestão de tarefas nem persistência.
- As imagens utilizadas são de URLs públicas (Unsplash). Substitua por imagens locais em `assets/` se preferir.
